yet another tetris clone written from scratch in c#
- has basic movement:
    left arrow  -> move left
    right arrow -> move right
    up arrow    -> rotate
    down arrow  -> move down
    space       -> move bottom
- has game over condition
- has row clearing
- "adapts" to terminal size
